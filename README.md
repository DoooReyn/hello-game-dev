# hello-game-dev

你好，这里是**游戏开发新手入门**。

我们将在这里积极收集有关游戏开发的资料，主要内容包括以下几个大类：

- 引擎
- 语言
- 编辑器/IDE
- 工具
- 资源
- 博客
- 论坛
- 邮件订阅
- 平台/渠道

每个大类在对应的目录下会细分，方便专注于某个方向的同学继续深入。

如有不足的地方，欢迎大家补充。
